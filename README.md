## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Usage

-   Publish URL: https://node-ghw2sq--3000.local.webcontainer.io/view
-   Back-end Endpoints:
    -   Get Post By Id (Example): https://node-ghw2sq--3000.local.webcontainer.io/posts/1
    -   Get All Posts: https://node-ghw2sq--3000.local.webcontainer.io/posts
-   Front-End Home page:
    -   https://node-ghw2sq--3000.local.webcontainer.io/view
